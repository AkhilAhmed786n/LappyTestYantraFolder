Accessifiers:--
-------------
1. Public 
2. Private
3. Protected
4. Default

Public:
-------
It is accessible within Variables, methods, classes
1. Public fields are Accessible within same class
2. Public fields are accessible within another class of same package
3. Public fields are accessible within another class of other package only by using Import statment of that class

Private:
--------
1. Private fields are accessible wihtin same class
2. Private fields are not accessible outside the declared class
3. Class can never be private
4. Private methods cant be inherited or Overridden

Protected:
----------
1. Protected fields can be accessible anywhere within class
2. Protected fields can be accessible within another class within same package.
3. Protected fields can access within another class within other package only by using inheritance.
4. Class can never be protected.
